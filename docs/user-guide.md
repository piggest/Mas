# Mas ユーザーガイド

**Mas**（Mac Area Screenshot）は、macOS向けのスクリーンショット撮影・編集アプリです。
キャプチャからアノテーション、テキスト認識まで、スクリーンショットに必要な機能をコンパクトにまとめています。

## 基本操作

Masはメニューバーに常駐するアプリです。メニューバーのアイコンから各機能にアクセスできます。

| 操作 | 動作 |
|------|------|
| アイコンをクリック | メニューを表示 |
| アイコンをダブルクリック | 範囲選択キャプチャを開始 |

メニューからは以下の操作が行えます：

- **全画面** — 画面全体をキャプチャ
- **範囲選択** — ドラッグで範囲を指定してキャプチャ
- **キャプチャ枠を表示** — 前回の範囲を枠として画面上に表示
- **ライブラリ** — キャプチャ履歴を一覧表示
- **開く...** — 画像ファイルを選択して編集

---

## キャプチャ方法

### 全画面キャプチャ

画面全体をキャプチャします。

- メニューから「全画面」を選択、または `⌘⇧3`

### 範囲選択キャプチャ

ドラッグで任意の範囲を指定してキャプチャします。

- メニューから「範囲選択」を選択、または `⌘⇧4`
- 画面上でドラッグして範囲を決定
- 前回の範囲を記憶しているので、同じ範囲を繰り返しキャプチャしやすくなっています

### キャプチャ枠表示

画面上に前回のキャプチャ範囲を枠として表示します。枠の位置やサイズをドラッグで調整してからキャプチャできます。

- メニューから「キャプチャ枠を表示」を選択、または `⌘⇧6`

### キャプチャ後の動作

キャプチャ後は自動的にエディターウィンドウが開きます。設定により以下の動作を制御できます：

- クリップボードへの自動コピー
- ファイルへの自動保存（PNG / JPEG）
- キャプチャ音の再生

## 編集ツール

エディターウィンドウでは、フローティングツールバーから各種編集ツールを使用できます。

### ペン

自由に線を描画します。色と太さを変更可能です。

### マーカー

半透明のハイライトを描画します。重要な箇所のマークアップに便利です。

### 矢印

矢印を描画します。注目ポイントの指示に使えます。

### 四角

四角形を描画します。範囲の強調に便利です。

### 丸

円形を描画します。四角と同様に範囲の強調に使えます。

### 文字

テキストを追加します。画像上の任意の位置をクリックしてテキストを入力できます。

### ぼかし

モザイク効果を適用します。個人情報や機密情報のマスキングに使えます。

### トリミング

スクリーンショットの範囲を切り出します。ドラッグで範囲を指定し、不要な部分を除去できます。

### 共通操作

- **色の変更** — ツールバーの色パレット（6色）から選択
- **太さの変更** — ツールバーのスライダーで調整（1〜30pt）
- **縁取り** — ツールバーのトグルで有効/無効を切り替え
- **元に戻す** — ツールバーのUndoボタンで最後のアノテーションを削除
- **選択・移動** — 移動ツールでアノテーションをクリックして選択、ドラッグで移動
- **削除** — アノテーション選択中にDeleteキーまたはツールバーの削除ボタン

## テキスト選択（OCR）

スクリーンショット内のテキストを認識し、選択・コピーできる機能です。

### 使い方

1. ツールバーから「テキスト選択」（虫眼鏡アイコン）を選択
2. OCRが自動的に実行され、テキスト領域が薄くハイライトされます
3. テキスト上をドラッグして、文字単位で選択できます
4. 「コピー」ボタンまたは `⌘C` で選択テキストをクリップボードにコピー
5. 画像の空白部分をクリックすると選択が解除されます

### 対応言語

- 日本語
- 英語

## ライブラリ

キャプチャ履歴を管理するウィンドウです。メニューから「ライブラリ」を選択して開きます。

### 機能

- **一覧表示** — サムネイル付きでキャプチャ履歴を表示
- **開く/閉じる** — エントリーをクリックしてエディターウィンドウを開閉
- **お気に入り** — 星アイコンでお気に入り登録。フィルターで絞り込み可能
- **削除** — 不要な履歴を削除
- **ピン** — エディターウィンドウの最前面表示を切り替え

## 設定

メニューの歯車アイコンから設定ウィンドウを開けます。

### 一般

| 設定項目 | 説明 |
|---------|------|
| クリップボードにコピー | キャプチャ後に自動コピー |
| ファイルに保存 | キャプチャ後に自動保存 |
| 保存先フォルダ | 保存先ディレクトリの指定 |
| 保存形式 | PNG / JPEG |
| JPEG品質 | 10〜100%で調整 |
| ピン機能 | 常にON / 最新のみON / デフォルトOFF |
| ドラッグ時に閉じる | ドラッグ成功時にウィンドウを自動で閉じる |
| マウスカーソルを含める | キャプチャにカーソルを含める |
| キャプチャ音 | キャプチャ時にサウンドを再生 |
| 開発者モード | 有効にすると「開発」タブが表示される |

### ショートカット

現在のキーボードショートカットを確認できます。

### 開発（開発者モード時のみ）

アノテーションデータなどを格納しているフォルダをFinderで開けます。

### 情報

アプリのバージョン情報を確認できます。

## キーボードショートカット

| ショートカット | 機能 |
|--------------|------|
| `⌘⇧3` | 全画面キャプチャ |
| `⌘⇧4` | 範囲選択キャプチャ |
| `⌘⇧6` | キャプチャ枠を表示 |
| `⌘O` | 画像ファイルを開く |

### エディターウィンドウ内

| ショートカット | 機能 |
|--------------|------|
| `Delete` / `Backspace` | 選択中のアノテーションを削除 |
| `⌘C` | 選択テキストをコピー（テキスト選択モード時） |

## その他

### 画像のドラッグ&ドロップ

エディターウィンドウの画像をドラッグして、他のアプリケーションにドロップできます。アノテーション付きの画像がそのまま渡されます。

### ウィンドウのピン固定

エディターウィンドウのピンアイコンをクリックすると、ウィンドウを最前面に固定できます。設定で初期状態を変更可能です。
